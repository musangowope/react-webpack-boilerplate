## Features

- React 16
- Webpack 4
- Babel 7
- Hot Module Replacement
- code style and linting: eslint & prettier
- HtmlWebpackPlugin

## Installation

- `git clone git@github.com:sjondegast/react-webpack-boilerplate.git`
- cd react-webpack-boilerplate
- npm install
- npm start
- visit `http://localhost:8080/`
